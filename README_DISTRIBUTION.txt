PV制作ツール 配布用README
作成日: 2026-04-30

■ 内容
このフォルダは、PV制作ツール本体です。
曲ごとのPV素材、歌詞、同期JSONを読み込み、歌詞焼きこみ動画を作成できます。

■ 起動方法
1. start_pv_tool.bat をダブルクリックします。
2. 初回起動時、必要なPythonライブラリを自動確認します。
3. 準備が終わるとブラウザで以下が開きます。

http://127.0.0.1:8767/pv_project_tool.html

■ 自動確認・自動インストール
start_pv_tool.bat は以下を確認します。

- Python 3.10以上
- ffmpeg
- Pythonライブラリ
  - opencv-python
  - numpy
  - pillow

Pythonライブラリが無い場合は、自動で pip install します。
Python本体やffmpegが無い場合は、wingetで入れるか確認します。
勝手にPCへソフトを追加しません。

■ 曲フォルダの置き方
曲ごとに以下のようなフォルダを作ります。

C:\Codex\PV\曲名
C:\Codex\PV\曲名\sauce

sauce フォルダには素材を入れます。

- PV動画: sauce\曲名 PV.mp4
- 歌詞TXT: sauce\曲名.txt

ツールが作る/使う主なファイルは曲フォルダ直下に置きます。

- pv_project.json
- pv_lyrics_timing.json
- lyrics_burn_preview.mp4
- 本番焼きこみ動画

■ 基本手順
1. 曲フォルダを選ぶ
2. PVと歌詞を選ぶ
3. 同期JSONを読み込む
4. 必要なら BOX細分化
5. 同じフォルダに保存
6. 本番焼きこみ

焼きこみプレビューは確認用です。
通常運用では、本番焼きこみだけでも使えます。

■ BOX細分化について
長い歌詞BOXは、焼きこみ時に表示が遅れたり詰まったりしやすくなります。
BOX細分化を押すと、長いBOXを短く分けます。
BOXの頭を守りやすくなるため、長い曲では推奨です。

■ 注意
- 元PVファイルは上書きしません。
- 焼きこみ結果は曲フォルダ側に出力されます。
- 曲フォルダに pv-tool フォルダを指定しないでください。
- 配布時はこのフォルダ一式をコピーしてください。

■ トラブル時
起動できない:
- start_pv_tool.bat をもう一度実行してください。
- Pythonまたはffmpegのインストール確認が出た場合は、内容を確認して進めてください。

焼きこみできない:
- 曲フォルダが正しく選ばれているか確認してください。
- pv_lyrics_timing.json が曲フォルダ直下にあるか確認してください。

PVが表示されない:
- sauce フォルダ内のPV動画を選び直してください。
- 生成済みの lyrics_burn_preview.mp4 ではなく、元PVを選んでください。
