PV制作ツール

場所:
C:\Codex\PV\pv-tool\PV制作ツール

起動:
start_pv_tool.bat

URL:
http://127.0.0.1:8767/pv_project_tool.html

できること:
- PVのプロジェクト情報を整理
- 動画プレビュー
- pv_lyrics_timing.json からシーン表を作成
- シーンごとのムード、画作りメモ、状態を編集
- pv_project.json と絵コンテMarkdownを書き出し

注意:
- このツールは元MP4を書き換えません。
- JSON書き出しはブラウザのダウンロードです。
- ブラウザ保存は localStorage です。
